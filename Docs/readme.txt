This download contains a Delphi unit that defines classes that
encapsulate 32 bit Windows resource files.

See install.html for installation instructions.

See user_guide.html for information on how to use the classes and for
sample code. (Requires the bullet_X.gif images files).

See changelog.txt for a version history.

See pad_file.xml for a brief description and contact information.

See resfile_fmt.html for a discussion of the resource file format and
the predefined resource types.
